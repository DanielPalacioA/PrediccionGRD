Instrucciones para ejecutar la aplicación
Para iniciar la aplicación en Django, sigue estos pasos:

Activa el entorno virtual incluido en el proyecto con el siguiente comando (Windows):

bash
venv/Scripts/activate
Navega a la carpeta que contiene manage.py:

bash
cd ProyectoGRD
Inicia el servidor de desarrollo con el comando:

bash
python manage.py runserver
Accede a la aplicación haciendo clic en la dirección del servidor que aparecerá en la terminal (generalmente http://127.0.0.1:8000/).
